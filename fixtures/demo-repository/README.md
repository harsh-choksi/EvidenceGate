# sourced-answer-demo

This fixture describes the demonstration repository used by EvidenceGate. The `incomplete-patch`
and `corrected-patch` directories are two deterministic candidate states for the same task.

- Incomplete: a superficially passing test, hard-coded sources, and no citation integrity.
- Corrected: Responses API web search, official-domain policy, source/annotation binding, and four
  security-focused behavior tests.
